# PTTEP-LinkMe
A modern solution for sharing contact details instantly and professionally.

## Description 
PTTEP LinkMe is a simple understanding cross-platform mobile app developed using Flutter which allows the targeted users, employees from [PTTEP](https://share.google/R9A2z3xIyyDuCxxyA) to share their contact details with others, whether networking at events, meeting clients, or even connecting wih colleagues easily in a digital way.

### Login Page  
This app uses multi-factor authentication (MFA) for user login session to ensure security. With assistance of Flutter package such as [flutter_web_auth](https://pub.dev/packages/flutter_web_auth) and some configuration done at PTTEP's Microsoft Azure account, it is possible to allow the staff to Login using their Microsoft account.  
![Login page screenshot of PTTEP LinkMe app](https://github.com/user-attachments/assets/3bdd8fc4-7612-4ba8-ac75-50239e041164)

### Home Page  
A glance to the app's home page, where it display the user's personal details such as name, job title, and some contact details in a complete way. Moreover, not only a QR code which brings scanner to a website version of the user's profile is displayed along, by clicking at the share icon button at the bottom right, users can take initiative to share their profile with others in several ways, such as copying their profile link, sharing via email and social medias, downloading as well as sharing the QR code.
